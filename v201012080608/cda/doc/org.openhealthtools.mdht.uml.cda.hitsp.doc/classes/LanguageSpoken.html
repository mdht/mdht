Language Spoken

[LanguageCommunication: templateId 2.16.840.1.113883.3.88.11.83.2]

  1. Conforms to IHE Language Communication template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.2.1)
  2. C154-[DE-2.01-1]: SHALL contain [1..1] languageCode, which SHALL be selected from ValueSet 2.16.840.1.113883.1.11.11526 Language DYNAMIC 200609
  3. C83-[DE-2.01-CDA-4]: SHALL contain [0..1] modeCode, which SHALL be selected from ValueSet 2.16.840.1.113883.1.11.12249 LanguageAbilityMode Value Set STATIC 1
    • Mode codes SHALL be appropriate to the type of language. Thus English, as spoken in the U.S. SHOULD use the code en-US and SHOULD only use mode codes for written and verbal communications. On the other hand, American Sign Language would be represented using the code sign-US, and would only use mode codes for signed communication.
  4. SHOULD contain [0..0] proficiencyLevelCode
  5. C83-[DE-2.01-CDA-1]: SHALL satisfy: Languages spoken shall be recorded using the <languageCommunication> infrastructure class associated with the patient. The <languageCommunication> element describes the primary and secondary languages of communication for a person.
  6. C154-[DE-2.01-2]: SHALL satisfy: Sign language is treated as a separate language.
