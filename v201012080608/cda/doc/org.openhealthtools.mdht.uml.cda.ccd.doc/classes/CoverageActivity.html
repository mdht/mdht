Coverage Activity

[Act: templateId 2.16.840.1.113883.10.20.1.20]

  1. Conforms to CDA Act
  2. CONF-36: SHALL contain [1..1] @classCode = "ACT"
  3. CONF-37: SHALL contain [1..1] @moodCode = "DEF"
  4. CONF-38: SHALL contain [1..*] id
  5. CONF-39, CONF-40: SHALL contain [1..1] statusCode/@code = "completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus STATIC V3NE08)
  6. CONF-41, CONF-42: SHALL contain [1..1] code/@code = "48768-6" Payment sources (CodeSystem: 2.16.840.1.113883.6.1 LOINC STATIC 2.26)
  7. CONF-43, CONF-45, CONF-46: SHALL contain [1..*] entryRelationship, such that it
    1. has @typeCode="COMP" COMP (has component)
    2. contains Policy Activity (templateId: 2.16.840.1.113883.10.20.1.26)
  8. CONF-47: SHALL satisfy: An alert observation contains one or more sources of information.
  9. CONF-44: MAY satisfy: entryRelationship contains sequenceNumber, which serves to prioritize the payment sources.
