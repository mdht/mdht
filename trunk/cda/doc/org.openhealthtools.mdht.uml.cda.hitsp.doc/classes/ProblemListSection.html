Problem List Section

[Section: templateId 2.16.840.1.113883.3.88.11.83.103]

The Problem List Section contains data on the problems currently being monitored for the patient.

  1. Conforms to IHE Active Problems Section template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.3.6)
  2. SHALL contain [1..*] entry, such that it
    1. contains Condition (templateId: 2.16.840.1.113883.3.88.11.83.7)
