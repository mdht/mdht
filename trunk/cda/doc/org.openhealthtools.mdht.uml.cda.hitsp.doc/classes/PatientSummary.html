Patient Summary

[ClinicalDocument: templateId 2.16.840.1.113883.3.88.11.32.1]

This Component describes the document content that summarizes a consumer's medical status for the purpose of health information exchange. While an EHR or PHR system can contain much more information, this Component only deals with the summary information to be exchanged between such systems as established as requirements described in AHIC Use Cases.

  1. Conforms to IHE Medical Document template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.1.1)
  2. Conforms to CCD Continuity Of Care Document template (templateId: 2.16.840.1.113883.10.20.1)
  3. C32-[CT1-1]: MAY contain [0..1] component, such that it
    1. contains Advance Directives Section (templateId: 2.16.840.1.113883.3.88.11.83.116)
  4. C32-[CT1-2]: MAY contain [0..1] component, such that it
    1. contains Allergies Reactions Section (templateId: 2.16.840.1.113883.3.88.11.83.102)
  5. C32-[CT1-3]: MAY contain [0..*] component, such that it
    1. contains Comment (templateId: 2.16.840.1.113883.3.88.11.83.11)
  6. C32-[CT1-4]: MAY contain [0..1] component, such that it
    1. contains Problem List Section (templateId: 2.16.840.1.113883.3.88.11.83.103)
  7. C32-[CT1-5]: MAY contain [0..1] component, such that it
    1. contains Encounters Section (templateId: 2.16.840.1.113883.3.88.11.83.127)
  8. C32-[CT1-7]: MAY contain [0..1] component, such that it
    1. contains Immunizations Section (templateId: 2.16.840.1.113883.3.88.11.83.117)
  9. C32-[CT1-9]: MAY contain [0..1] component, such that it
    1. contains Payers Section (templateId: 2.16.840.1.113883.3.88.11.83.101)
  10. C32-[CT1-11]: MAY contain [0..1] component, such that it
    1. contains Medications Section (templateId: 2.16.840.1.113883.3.88.11.83.112)
  11. C32-[CT1-13]: MAY contain [0..1] component, such that it
    1. contains Plan Of Care Section (templateId: 2.16.840.1.113883.3.88.11.83.124)
  12. C32-[CT1-14]: MAY contain [0..1] component, such that it
    1. contains IHE Pregnancy History Section (templateId: 1.3.6.1.4.1.19376.1.5.3.1.1.5.3.4)
  13. C32-[CT1-15]: MAY contain [0..1] component, such that it
    1. contains Surgeries Section (templateId: 2.16.840.1.113883.3.88.11.83.108)
  14. C32-[CT1-16]: SHOULD contain [1..*] supportHeaders, such that it
    1. contains Support
  15. C32-[CT1-17]: MAY contain [0..1] component, such that it
    1. contains Vital Signs Section (templateId: 2.16.840.1.113883.3.88.11.83.119)
  16. C32-[CT1-18]: MAY contain [0..1] component, such that it
    1. contains Diagnostic Results Section (templateId: 2.16.840.1.113883.3.88.11.83.122)
  17. C32-[CT1-6]: MAY satisfy: Contains 0..* HealthcareProvider in cda:documentationOf/cda:serviceEvent/cda:performer
  18. C32-[CT1-8]: SHALL satisfy: Contains 0..* InformationSource in ancestor-or-self::./cda:author[1]
  19. C32-[CT1-10]: SHOULD satisfy: Contains 0..* LanguageSpoken in cda:recordTarget/cda:patientRole/cda:patient/cda:languageCommunication
  20. C32-[CT1-12]: SHALL satisfy: Contains 1..1 Person Information in cda:recordTarget/cda:patientRole
