Problem Entry

[Observation: templateId 1.3.6.1.4.1.19376.1.5.3.1.4.5]

  1. Conforms to CCD Problem Observation template (templateId: 2.16.840.1.113883.10.20.1.28)
  2. SHALL contain [1..1] text
  3. SHALL contain [1..1] value, where its data type is CD
