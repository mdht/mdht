Payers Section

[Section: templateId 1.3.6.1.4.1.19376.1.5.3.1.1.5.3.7]

The Payers section contains data on the patient's payers, whether a 'third party' insurance, self-pay, other payer or guarantor, or some combination.

  1. Conforms to CCD Payers Section template (templateId: 2.16.840.1.113883.10.20.1.9)
  2. SHOULD contain [1..*] entry, such that it
    1. contains Coverage Entry (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.17)
