Vital Sign Observation

[Observation: templateId 1.3.6.1.4.1.19376.1.5.3.1.4.13.2]

  1. SHALL conform to CCD Result Observation template (templateId: 2.16.840.1.113883.10.20.1.31)
  2. SHALL conform to Simple Observation template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.13)
  3. SHALL contain [1..1] code (CodeSystem: 2.16.840.1.113883.6.1 LOINC STATIC)
  4. MAY contain [0..*] interpretationCode
    • The interpretation code may be present to provide an interpretation of the vital signs measure (e.g., High, Normal, Low, et cetera).
  5. MAY contain [0..*] methodCode
    • The method code element may be present to indicate the method used to obtain the measure. Note that method used is distinct from, but possibly related to the target site.
  6. Contains [1..1] statusCode
    • The observations have all been completed.
  7. MAY contain [0..*] targetSiteCode
    • The target site of the measure may be identified in the targetSiteCode element (e.g., Left arm [blood pressure], oral [temperature], et cetera).
  8. SHALL contain [1..1] value, where its data type is PQ
