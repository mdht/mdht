Comment

[Act: templateId 1.3.6.1.4.1.19376.1.5.3.1.4.2]

  1. Conforms to CDA Act
  2. SHALL contain [1..1] @classCode = "ACT"
  3. SHALL contain [1..1] @moodCode = "EVN"
  4. SHALL contain [1..1] code/@code = "48767-8" Annotation Comment (CodeSystem: 2.16.840.1.113883.6.1 LOINC STATIC 2.26)
  5. SHALL contain [0..1] statusCode/@code = "completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus STATIC V3NE08)
  6. SHALL contain [0..1] text
