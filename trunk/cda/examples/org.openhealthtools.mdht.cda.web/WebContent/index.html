MDHT Clinical Document Architecture (CDA) Document Validation Services

The site provides validation services for CDA documents using the MDHT CDA Tooling solution and the complete source code is available on the MDHT Subversion. The web page uses a web service to provide XML validation content which is then rendered using XSLT. There are currently two displays of validation results, Tree and Table View.
