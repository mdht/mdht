A note about these Installations

These installation will require you to download and install several file over the internet.  If you are behind a fire wall or on a VPN your overall download and installation time will be impacted.  If possible, attempt an installation when you have sufficient bandwidth and can dedicate up to an hour to complete the installation.  You must also have administration rights or the equivalent to down load and save several zipped files.

Milestone Builds

Milestone builds represent planned iteration points in our development process. User testing and feedback is solicited to validate readiness for promotion to release status. Milestone builds may contain pre-release software that is pending approval for redistribution under the Open Health Tools Software User Agreement.

Recommended procedure is to install using the update site. Instructions at the top of this page.

CDA Tools

Milestone M1 6th October 2009 (update site ZIP)

IMPORTANT NOTE: If you have ANY test build installed prior to 1st July 2009, then you MUST uninstall it prior to using the update site to install this release candidate. The Eclipse feature IDs have changed. If you do not uninstall the prior features, then you will have duplicate functionality where all menus, properties tabs, etc. will appear twice. For guidance, see screen image showing selection and uninstall of previous build.
