Model-Driven Health Tools (MDHT)

Papyrus Update Site Installation Instructions

All project builds are intended to be installed in your Eclipse platform using the Eclipse Update Manager, accessed using the Help > Software Updates... menu. The update sites are currently located on a SourceForge server where we have access to web file directories (unavailable on the OHT server). The update sites contain release and milestone builds, but not integration development builds. If you are unfamiliar with use of the update manager:

Install from the following update sites URLs:

  • For MDHT Tools: http://oht-modeling.sourceforge.net/mdht/updates
  1. Add an update site location (screen image)

  2. Select your desired installation feature options based on your platform

    CDA

    • UML for CDA, All Users Platform
    • UML for CDA, RSM Platform
    • UML for CDA, Papyrus Platform

    RIM

    • UML for RIM, All Users Platform
    • UML for RIM, RSM Platform
    • UML for RIM, Papyrus Platform
