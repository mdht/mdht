A note about Papyrus

The version of Papyrus being installed is a modified version of the eclipse Papyrus project M3 release in order to work with the MDHT UML extensions.  The modification fixes a reported issue that has yet to be resolved and committed to the Papyrus build.  The details of issue and resolution can be found on the Papyrus issue tracker here Papyrus Issue 291818

A note about this Installation

This installation will require you to download and install several file over the internet.  If you are behind a fire wall or on a VPN your overall download and installation time will be impacted.  If possible, attempt the installation when you have sufficient bandwidth and can dedicate up to an hour to complete the installation.  You must also have administration rights or the equivalent to down load and save several zipped files.

Down load and Extract the MDHT Papyrus

After you have down loaded the complete application and extracted all of its contents into a sub directory, you are ready to start the application. Browse to the eclipse directory and start the eclipse application. Congratulations, you have successfully installed the MDHT Papyrus application and are now ready to install the MDHT CDA plugins.
