Installation Guides for Papyrus

A note about Papyrus

The version of Papyrus being installed is a modified version of the eclipse Papyrus project M3 release in order to work with the MDHT UML extensions.  The modification fixes a reported issue that has yet to be resolved and committed to the Papyrus build.  The details of issue and resolution can be found on the Papyrus issue tracker here Papyrus Issue 291818

  • All-In-One Installation Guide

    All-In-One Installation will down load a completely configured eclipse application. Use this option if you do not have eclipse already installed or want to have a new instance of eclipse.

  • Step by Step Installation Guide

    Step by Step Installation will walk you through the individual component installations. Use this option if you have an existing eclipse or want to understand the required components.
