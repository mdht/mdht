MDHT Papyrus Installation Guide

This document contains the necessary steps required to successfully install the Open Source UML application Papyrus. Papyrus is currently part of the eclipse Model Driven Tools project (See http://wiki.eclipse.org/MDT/Papyrus ). The installation will

1.       Install a new version of eclipse version 3.5 (also known as Galileo)

2.       Install Eclipse Model Development Tools (MDT)

3.       Install Eclipse Modeling Framework (EMF)

4.       Install Eclipse Subversion

5.       Install Eclipse Extended Editing Framework

6.       Install Papyrus for MDHT M3

A note about Papyrus

The version of Papyrus being installed is a modified version of the eclipse Papyrus project M3 release in order to work with the MDHT UML extensions.  The modification fixes a reported issue that has yet to be resolved and committed to the Papyrus build.  The details of issue and resolution can be found on the Papyrus issue tracker here Papyrus Issue 291818

A note about this Installation

This installation will require you to download and install several file over the internet.  If you are behind a fire wall or on a VPN your overall download and installation time will be impacted.  If possible, attempt the installation when you have sufficient bandwidth and can dedicate up to an hour to complete the installation.  You must also have administration rights or the equivalent to down load and save several zipped files.

Step 1 Install Eclipse 3.5 for Java Developers

Using the following link, down load and extract Eclipse Java Version 3.5 for Windows (See eclipse packages   for a description of the version to be installed) Be aware that this down load can take a while

http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/galileo/SR1/eclipse-java-galileo-SR1-win32.zip

Start the eclipse.exe application

 

Step 2 Install Eclipse Model Development Tools Project (MDT)

Using the following menu options

                Help->Install New Software..

Add the MDT software site using the following location

http://download.eclipse.org/modeling/mdt/updates/releases/

Install the following two components from the MDT project

MDT OCL SDK 1.3.0
MDT UML 2 SDK 3.0.1

Please Restart Eclipse

 

Step 3 Install Eclipse Modeling Framework Project (EMF)

Using the following menu options

                Help->Install New Software..

Add the EMF software site using the following location

http://download.eclipse.org/modeling/emf/updates/releases/

Install the following three components from the EMF project

EMF Query SDK 1.3
EMF SDK 2.5
EMF Transaction 1.3.1

Please Restart Eclipse

Step 4 Install Subversion Plugin

Using the following menu options

                Help->Install New Software..

Add the Subversion site using the following location

http://subclipse.tigris.org/update_1.6.x

Install the following three components from the Subversion plugin

Core SVNKit Library
Optiona JNA Library
Subclipse

Please Restart Eclipse
