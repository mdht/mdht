Model-Driven Health Tools (MDHT)

MDHT Papyrus Installation Guide

This document contains the necessary steps required to successfully install the Open Source UML application Papyrus. Papyrus is currently part of the eclipse Model Driven Tools project (See http://wiki.eclipse.org/MDT/Papyrus ). The installation will

  • Install a new version of eclipse version 3.5 (also known as Galileo)
  • Install Eclipse Model Development Tools (MDT)
  • Install Eclipse Modeling Framework (EMF)
  • Install Eclipse Subversion
  • Install Eclipse Extended Editing Framework
  • Install Papyrus for MDHT M3
  • Install MDHT for Papyrus using Eclipse Update Site

Step 1 Install Eclipse 3.5 (Galileo) for Java Developers

Using the following link, down load and extract Eclipse Java Version 3.5 for Windows (See eclipse packages   for a description of the version to be installed) Be aware that this down load can take a while

Eclipse 3.5 (Galileo) for Java Developers

Start the eclipse.exe application

 

Step 2 Install Eclipse Model Development Tools Project (MDT)

Using the following menu options Help->Install New Software..

Add the MDT software site using the following location

http://download.eclipse.org/modeling/mdt/updates/releases/

Install the following two components from the MDT project

  • MDT OCL SDK 1.3.0
  • MDT UML 2 SDK 3.0.1

Please Restart Eclipse

 

Step 3 Install Eclipse Modeling Framework Project (EMF)

Using the following menu options Help->Install New Software..

Add the EMF software site using the following location

http://download.eclipse.org/modeling/emf/updates/releases/

Install the following three components from the EMF project

  • EMF Query SDK 1.3
  • EMF SDK 2.5
  • EMF Transaction 1.3.1

Please Restart Eclipse

Step 4 Install Subversion Plugin

Using the following menu options Help->Install New Software..

Add the Subversion site using the following location

http://subclipse.tigris.org/update_1.6.x

Install the following three components from the Subversion plugin

  • Core SVNKit Library
  • Optiona JNA Library
  • Subclipse

Please Restart Eclipse

Using the following link, down load and save (you do not need to extract) the EEF Installation Site

  • Extended Editing Framework Incubation

Step 6 Download and Install Eclipse Papyrus Application M3 by MDHT

Using the following link, down load and save (you do not need to extract) the Eclipse Papyrus Application Installation Site

Using the following menu options

                Help->Install New Software..

  • Additional Tools
  • BackBone

  • UML Editor

Please Restart Eclipse.

Almost Done! Follow the instructions for installing the MDHT CDA plugin at the following location Papyrus Eclipse Update Site Installation to complete the installation.
