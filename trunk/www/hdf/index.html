Overview

HDF Tools is a component within the Model Driven Health Tools (MDHT) project. This component will develop a UML approach for designing and implementing HL7 v3 RIM-based domain and message models. The modeling capabilities include:

  • A UML profile with stereotypes that extend models with HL7 metadata
  • Plug-ins and custom Ant tasks for importing and exporting HL7 MIF representation to/from UML models
  • Generation of XML Schemas from HL7 styled UML models
  • Other components under development or part of future project plans...

A separate project component is dedicated to development of CDA Tools that are specialized for designing and implementing CDA templates.
