UML Core Components

Back to Model-Driven Healthcare Tools home page

This project provides a set of UML Core components that support the model-driven analysis, design, and implementation of healthcare standards and applications. A more detailed description is available on our project wiki, but examples of these foundation components include:

  • A UML profile with stereotypes that extend models with HL7 metadata
  • Plug-ins and custom Ant tasks for importing and exporting HL7 MIF representation from UML models
  • Generation of XML Schemas from HL7 styled UML models
  • Other components under development or part of future project plans...

A separate project component is dedicated to development of CDA Tools that are built on these foundation components.
