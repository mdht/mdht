The Project Set Files (PSF) in this folder are provided to simplify the process 
for developers to check out a set of projects to initialize a new workspace.
Several variations may be available as different .psf files, each of which
includes a different set of projects.

To use a .psf file, copy of of these files to your computer file system, then
use the wizard to select the .psf file.

File > Import...
Team > Team Project Set
