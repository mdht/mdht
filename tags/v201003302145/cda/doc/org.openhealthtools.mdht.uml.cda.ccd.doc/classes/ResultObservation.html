  1. Conforms to CDA Observation
  2. SHALL contain [1..1] @moodCode = "EVN"
  3. SHALL contain [1..*] id
  4. SHOULD contain [1..1] effectiveTime
  5. SHALL contain [1..1] statusCode
  6. SHALL contain [1..1] code
  7. MAY contain [0..1] methodCode
  8. SHOULD contain [0..*] interpretationCode
  9. SHALL contain [1..1] value
  10. SHOULD satisfy: The value for 'code' SHOULD be selected from LOINC (codeSystem 2.16.840.1.113883.6.1) or SNOMED CT (codeSystem 2.16.840.1.113883.6.96), and MAY be selected from CPT-4 (codeSystem 2.16.840.1.113883.6.12).
  11. SHALL satisfy: The methodCode SHALL NOT conflict with the method inherent in code
  12. SHALL satisfy: Where value is a physical quantity, the unit of measure SHALL be expressed using a valid Unified Code for Units of Measure (UCUM) expression.
  13. SHOULD satisfy: Contain one or more referenceRange to show the normal range of values for the observation result
  14. SHALL satisfy: SHALL NOT contain referenceRange / observationRange / code, as this attribute is not used by the HL7 Clinical Statement or Lab Committee models.
  15. SHALL satisfy: Contains one or more sources of information.
