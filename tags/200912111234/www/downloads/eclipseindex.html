Model-Driven Health Tools (MDHT)

Installation Guides for Eclipse 3.5 (Galileo)

  • All-In-One Installation Guide

    All-In-One Installation will down load a completely configured eclipse application. Use this option if you do not have eclipse already installed or want to have a new instance of eclipse.

  • Eclipse Software Update Site Installation Guide

    Eclipse Software Update Site Installation uses the built in software installation feature from Eclipse. You must have at a minimum the Eclipse IDE for Java Developers (92 MB) installed.

  • Step by Step Installation Guide

    Step by Step Installation will walk you through the individual component installations. Use this option if you want to understand the required components.
