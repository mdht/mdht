Model-Driven Health Tools (MDHT)

Subversion is a free/open-source version control system. That is, Subversion manages files and directories, and the changes made to them, over time. This allows you to recover older versions of your data, or examine the history of how your data changed. In this regard, many people think of a version control system as a sort of time machine.

Install Subversion

Install Subversion Provider

Using the following menu options Help->Install New Software..

Add the Subversion Provider site using the following location

http://download.eclipse.org/technology/subversive/0.7/update-site/

Install the following three components from the Subversion plugin

Subversive SVN Team Provider Plugin (Incubation)

  • Subversive SVN JDT Ignore Extensions (Optional) (Incubation)
  • Subversive SVN Team Provider (Incubation)
  • Subversive SVN Team Provider Localization (Optional) (Incubation)

Press Next and Finish

Install Subversion Connector

Using the following menu options Help->Install New Software..

Add the Subversion Connector site using the following location

http://www.polarion.org/projects/subversive/download/eclipse/2.0/update-site/

Install the following two components from the Subversion plugin

Subversive SVN Connectors

  • Subversive SVN Connectors
  • SVNKit 1.3.0 (or latest) Implementation

Press Next and Finish

Please Restart Eclipse
