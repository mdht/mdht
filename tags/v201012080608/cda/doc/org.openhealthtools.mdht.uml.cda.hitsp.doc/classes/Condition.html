Condition

[Act: templateId 2.16.840.1.113883.3.88.11.83.7]

  1. Conforms to IHE Problem Concern Entry template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.5.2)
  2. SHALL contain [1..*] entryRelationship, such that it
    1. has @typeCode="SUBJ" SUBJ (has subject)
    2. contains Condition Entry
