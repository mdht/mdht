Result

[Observation: templateId 2.16.840.1.113883.3.88.11.83.15]

  1. Conforms to CCD Result Observation template (templateId: 2.16.840.1.113883.10.20.1.31)
  2. Conforms to IHE Simple Observation template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.13)
  3. SHALL contain [1..1] code
  4. SHALL contain [1..1] effectiveTime
