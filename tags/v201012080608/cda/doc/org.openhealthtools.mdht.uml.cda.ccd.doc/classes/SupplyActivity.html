Supply Activity

[Supply: templateId 2.16.840.1.113883.10.20.1.34]

  1. Conforms to CDA Supply
  2. SHALL contain [1..*] id
  3. SHOULD contain [1..1] statusCode
  4. MAY contain [0..1] entryRelationship, such that it
    1. contains Medication Status Observation (templateId: 2.16.840.1.113883.10.20.1.47)
  5. MAY contain [0..*] entryRelationship, such that it
    1. has @typeCode="SUBJ" SUBJ (has subject)
    2. contains Fulfillment Instruction (templateId: 2.16.840.1.113883.10.20.1.43)
  6. SHOULD contain [1..1] effectiveTime
