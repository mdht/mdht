Encounter Entry

[Encounter: templateId 1.3.6.1.4.1.19376.1.5.3.1.4.14]

  1. Conforms to CDA Encounter
  2. SHALL contain [1..1] @classCode = "ENC"
  3. SHOULD contain [0..1] code (CodeSystem: 2.16.840.1.113883.5.4 ActEncounterCode STATIC)
    • Developers should take care to check that rational combinations of encounter.code and encounter.moodCode are used , but this profile does not restrict any combination.
  4. SHALL contain [1..*] id
  5. SHALL contain [1..1] text
