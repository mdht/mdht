Vital Sign Observation

[Observation: templateId 1.3.6.1.4.1.19376.1.5.3.1.4.13.2]

  1. Conforms to CCD Result Observation template (templateId: 2.16.840.1.113883.10.20.1.31)
  2. Conforms to Simple Observation template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.13)
  3. SHALL contain [1..1] code (CodeSystem: 2.16.840.1.113883.6.1 LOINC STATIC 2.26)
  4. SHALL contain [1..1] value, where its data type is PQ
  5. MAY contain [0..*] interpretationCode
  6. MAY contain [0..*] methodCode
  7. MAY contain [0..*] targetSiteCode
