Purpose Section

[Section: templateId 2.16.840.1.113883.10.20.1.13]

Represents the specific reason for which the summarization was generated, such as in response to a request.

The general use case does not require a purpose. Purpose should be utilized when the CCD has a specific purpose such as a transfer, referral, or patient request.

  1. Conforms to CDA Section
  2. SHALL contain [1..1] code/@code = "48764-5" Summary purpose (CodeSystem: 2.16.840.1.113883.6.1 LOINC STATIC 2.26)
  3. SHALL contain [1..1] title
  4. SHOULD contain [1..*] entry such that it
    1. SHALL contain [1..1] Purpose Activity (templateId: 2.16.840.1.113883.10.20.1.30)
  5. SHALL contain [1..1] text
  6. SHOULD satisfy: Contains a case-insensitive language-insensitive string containing 'purpose'.
